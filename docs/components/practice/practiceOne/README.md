### 內置搜索：
只能搜索h2 & h3標題
::: tip 設定:
```js
// 設定內置搜索
search: true, // false就會不顯示,不設定的話默認為true
```
補充：  
對內置的搜索進行最多結果數量的限制
```js
searchMaxSuggestions: 3, // 限制搜索最多數量為3個
```
